Selectable tabs and iframe update widget

This is a widget for updating an iframe from URLs stored in tab container.

This is the source code:

html code for the list Container:

<div class="menu" id="proList">

   <p class="containerTitle" id="feedback">
     <span>Now Showing: </span> <span class="select-result">none</span>.
  </p>
  <div class="menu" id="proListContainer">
     <ol class="selectable">
        <li class="projectTab">TabName/li>
        <li class="projectTab">Another</li>
        <li class="projectTab">Another</li>
        <li class="projectTab">Another</li>
        <li class="projectTab">Another</li>
        <li class="projectTab">Another</li>
        <li class="projectTab">Another</li>
        <li class="projectTab">Another</li>
        <li class="projectTab">Another</li>
        <li class="projectTab">Another</li>
        <li class="projectTab">Another</li>
     </ol>
  </div>

</div>

The first span displays the name of the container, and the second span is updated every time that a tab is clicked. Further, the URL displayed in the iframe is also updated every time a tab is clicked.

html code for the corresponding iframe:

<iframe class="menu" id="proExp" src=""></iframe>

javaScript code for triggering the update of the iframe:

<script>
  $(".projectTab").click(function () {
     var name = $(this).text();
     var sites = ["URL1","URL2","..."];
     $(".select-result").text(name);
     $('#proExp')[0].src = sites[$(this).index()];
  });
</script>

The sites array corresponds to the index of the tabs displayed in the container.

For manipulating the container div position you will have to change the id proLits.

For manipulating the iframe div position you will have to change the id proExp.

For implementing complete functionality of the widget you will have to import the jquery library in your html file:

<script src="http://code.jquery.com/jquery.min.js"></script>

In this webpage you can see a double example of the selectable tabs widget.

The upper left box is an example of box with feedback, and the lower left box is an example of a container with no feedback. You can see that it does not matter which container tap is clicked, the feedback always updates.
